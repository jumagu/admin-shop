# Admin Shop

Clothing store built with Vue.js using the Composition API. It is a simple application that only provides authentication module and product management (creation and update).

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
